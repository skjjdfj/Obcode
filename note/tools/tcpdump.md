## 基本使用

显示网络接口：
```bash
ip a s
```

**参数**

| 命令              | 解释                              |
| --------------- | ------------------------------- |
| `tcpdump -i 接口` | 捕获特定网络接口上的数据包                   |
| `tcpdump -w 文件` | 将捕获的数据包写入文件                     |
| `tcpdump -r 文件` | 从文件中读取捕获的数据包                    |
| `tcpdump -c 计数` | 捕获特定数量的数据包                      |
| `tcpdump -n`    | 不解析 IP 地址                       |
| `tcpdump -nn`   | 不解析 IP 地址，也不解析协议号               |
| `tcpdump -v`    | 详细显示;可以使用 `-vv` 和 `-vvv` 增加详细程度 |
